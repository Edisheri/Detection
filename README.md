# Диагностика лёгочных заболеваний с помощью нейронных сетей

Готовое приложение для классификации рентгеновских снимков грудной клетки (Норма / Пневмония / COVID-19). При первом запуске автоматически подгружаются данные и обучается модель — ничего настраивать не нужно.

## Запуск (всё из коробки)

```bash
cd Med
pip install -r requirements.txt
python run.py
```

Откроется браузер с веб-интерфейсом. Загрузите рентген-снимок — получите результат и вероятности по классам.

При **первом** запуске программа сама:
1. Скачивает реальные снимки из открытого репозитория (COVID-19 и пневмония)
2. Обучает нейросеть (ResNet18)
3. Запускает приложение

Если интернет недоступен, создаётся демо-набор и модель обучается на нём (для проверки работы).

## Дополнительно

- **Только обучить модель заново:**  
  Положите свои данные в `data/chest_xray/train/` (папки: `Normal`, `Pneumonia`, `COVID-19`), затем:
  ```bash
  python train.py
  ```
- **Предсказание по файлу из терминала:**
  ```bash
  python predict.py путь/к/снимку.jpg
  ```

## Структура проекта

- `run.py` — точка входа: подготовка (при необходимости) + запуск веб-приложения
- `app.py` — веб-интерфейс (Streamlit)
- `setup.py` — автоматическая загрузка данных и первое обучение
- `train.py` — обучение модели
- `predict.py` — предсказание по одному изображению
- `config.py` — пути и параметры
- `src/` — модель, датасет, инференс

Результаты носят справочный характер и не заменяют заключение врача.
